Wanderlust

Wanderlust is a full-stack travel planning web application that allows users to explore, list, and review travel destinations. Built using HTML, CSS, Bootstrap, Node.js, Express, and MongoDB Atlas, it features image uploads via Cloudinary, secure user authentication with Passport.js, and interactive map views using OpenCage and OpenStreetMap. Key functionalities include CRUD operations on listings and reviews, user login and signup, and the ability to view destination locations directly on the map.
